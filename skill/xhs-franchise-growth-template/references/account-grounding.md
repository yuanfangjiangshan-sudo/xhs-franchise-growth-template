# Account Grounding

Use account data before generating performance-aware recommendations.

## Preferred Inputs

Ask the user for one of these:

- recent Xiaohongshu Creator Center data
- exported note performance table
- screenshots of recent note performance
- manually pasted data for recent posts
- a short account summary if data access is unavailable

## Minimum Useful Data

For each recent note, try to collect:

- publish date
- title/topic
- impressions or views
- likes
- saves
- comments
- follows
- private-message or keyword signals, if available

## Diagnosis Questions

Answer briefly:

- Which recent topic had the strongest signal?
- Which content structure seems to work?
- Which audience pain appears most active?
- Which titles or covers underperformed?
- What data is missing?

## If Data Is Missing

Use this wording:

> 当前没有读取到账号历史内容和发布数据，本次只能作为策略样稿；要做真正的账号优化，需要提供主页、创作者中心截图、导出数据或近期笔记表现。

Do not pretend to have read account performance.
